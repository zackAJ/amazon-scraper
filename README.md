# Scrape Amazon Products

### Live Demo: https://amazon-scrape.vercel.app/

## Instalation

1- Clone repo:
```
git clone https://github.com/zackAJ/amazon-scraper
```

2- Install dependencies:
```
npm install
```

3- Run project:
```
npm start
```

4- Open localhost at port 3000:
```
http://localhost:3000
```
